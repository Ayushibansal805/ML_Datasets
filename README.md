📊 Customer Segmentation using Machine Learning
📌 Overview
This project focuses on customer segmentation by applying supervised machine learning models to classify customers into distinct groups based on demographic and behavioral data. The goal is to enable data-driven marketing and personalization strategies.
🎯 Objectives
Identify meaningful customer segments
Compare multiple ML models for classification
Optimize model performance using tuning techniques
Evaluate model reliability using multiple metrics
📂 Dataset
Contains customer demographic and behavioral attributes
Includes categorical and numerical features
Preprocessed for model compatibility
⚙️ Tech Stack
Python
Pandas, NumPy
Scikit-learn
XGBoost
Seaborn, Matplotlib
🧹 Data Preprocessing
Label Encoding for categorical variables
Handling missing values
Feature scaling (for SVM)
Train-test split (80-20)
🤖 Models Implemented
Logistic Regression
Random Forest
XGBoost
Support Vector Machine (SVM)
📊 Evaluation Metrics
Accuracy
F1 Score
Confusion Matrix
📉 Visualization
Model comparison using Seaborn bar plots
Confusion matrix heatmaps
🚀 Key Insights
XGBoost outperforms other models on tabular data
Ensemble methods capture complex patterns effectively
Proper preprocessing significantly improves performance
📌 Conclusion
Machine learning-based segmentation enables businesses to better understand customer behavior and improve targeted marketing strategies.

🏎️ Formula 1 Strategy Prediction using Machine Learning
📌 Overview
This project analyzes Formula 1 race strategy data to predict key decisions such as tire selection and pit stops, as well as explore lap time prediction using regression models.
🎯 Objectives
Predict race strategy decisions (classification)
Analyze lap time performance (regression)
Compare multiple ML models
Improve model accuracy through tuning and feature engineering
📂 Dataset
F1 race telemetry and strategy dataset
Includes lap data, tire usage, degradation metrics, and race progress
Minimal missing values handled during preprocessing
⚙️ Tech Stack
Python
Pandas, NumPy
Scikit-learn
XGBoost
Seaborn, Matplotlib
🧹 Data Preprocessing
Handling missing values (row removal due to minimal NaNs)
Feature scaling (for SVM)
Train-test split
🤖 Models Implemented
🔹 Classification
Logistic Regression
Random Forest
XGBoost
Support Vector Machine (SVM)
🔹 Regression
Linear Regression
XGBoost Regressor
📊 Evaluation Metrics
Classification
Accuracy
F1 Score
Confusion Matrix
📉 Visualization
Model comparison bar plots (Seaborn)
Feature importance analysis
🚀 Key Insights
Strategy prediction (classification) is more reliable than lap time prediction
Feature engineering significantly improves model performance
Tree-based models are highly effective for structured racing data
📌 Conclusion
Machine learning can effectively model Formula 1 strategies, providing insights into race decisions and performance trends. Classification tasks yield more stable and interpretable results compared to regression-based lap time prediction.
